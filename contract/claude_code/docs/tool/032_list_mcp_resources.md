# Tool: ListMcpResourcesTool

List resources exposed by connected MCP servers.

### Category

MCP Resources

### Permission Required

No

### Description

Lists all resources exposed by currently connected MCP servers. Resources are
named data items (files, database records, API responses) that MCP servers make
available for reading. Returns resource URIs, names, and descriptions.

### Parameters

None.
