# Parameter: disable_growthbook

### Forms

| Form | Value |
|------|-------|
| Env Var | `DISABLE_GROWTHBOOK` |

### Type

boolean (presence-activated)

### Default

Not set (GrowthBook enabled)

### Description

Disables GrowthBook feature-flag fetching. Prevents Claude Code from retrieving
A/B test and feature flag configurations from Anthropic's GrowthBook instance.

### Since

pre-v1.0 (unverified)
