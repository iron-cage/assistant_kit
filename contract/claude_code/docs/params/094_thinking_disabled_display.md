# thinking_disabled_display

Controls display behavior when thinking/extended thinking is disabled.

### Forms

| | Value |
|-|-------|
| CLI Flag | — |
| Env Var | — |
| Config Key | `thinking.disabled.display` |

### Type

string

### Default

Binary default (unspecified)

### Since

v2.1.181

### Description

Controls how Claude Code displays or hides the thinking indicator when extended
thinking is disabled for the session. Allows customizing the UI behavior around
thinking blocks.
