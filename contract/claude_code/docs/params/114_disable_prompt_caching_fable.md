# Parameter: disable_prompt_caching_fable

### Forms

| Form | Value |
|------|-------|
| Env Var | `DISABLE_PROMPT_CACHING_FABLE` |

### Type

boolean (presence-activated)

### Default

Not set (prompt caching enabled for Fable)

### Description

Disables prompt caching for Fable models only.

### Since

v2.1.170+ (unverified) — Fable 5 introduced v2.1.170
