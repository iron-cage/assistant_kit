# Parameter: disable_installation_checks

### Forms

| Form | Value |
|------|-------|
| Env Var | `DISABLE_INSTALLATION_CHECKS` |

### Type

boolean (presence-activated)

### Default

Not set (installation checks enabled)

### Description

Disables installation-related warnings shown at startup (e.g., outdated Node.js
version, permissions issues).

### Since

pre-v1.0 (unverified)
