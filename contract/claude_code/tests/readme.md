# Tests

Test suite for the `claude_code` contract crate.

### Scope

- **Purpose**: Host all test infrastructure validating `claude` binary contracts and fault classification specs.
- **Responsibility**: Index of test directories covering behavior hypothesis tests and test spec planning documents.

### Responsibility Table

| Directory | Responsibility |
|-----------|----------------|
| `behavior/` | Behavior hypothesis invalidation tests (B1–B24 + B16h); invokes real `claude` binary |
| `docs/` | Test spec planning documents (FT-N) for fault classification |
