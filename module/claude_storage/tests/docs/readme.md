# Test Documentation

### Scope

- **Purpose**: Test-facing documentation mirroring and extending the `docs/` hierarchy.
- **Responsibility**: Test planning, test case indexes, and test-specific documentation organized parallel to `docs/`.
- **In Scope**: All test specification and planning documents for the `claude_storage` crate.
- **Out of Scope**: Automated test implementations (→ `tests/*.rs`), manual testing plans (→ `tests/manual/`).
