# Parameter :: 42. `min_7d::`

Filters the `.usage` table to hide rows where `7d Left` is below the specified percentage threshold.

- **Type:** `f64`
- **Default:** `0` (no filter)
- **Constraints:** `0` to `100` (inclusive)
- **Commands:** [`.usage`](../command/006_usage.md#command--9-usage)
- **Purpose:** Show only accounts whose 7d Left is at or above the given percentage.
- **Group:** Display Control

**Behavior:** Rows where `7d Left < threshold` are hidden. The threshold is inclusive — a row with `7d Left = threshold` is shown. Rows with `7d Left = —` (no valid quota data) are also hidden when any threshold > 0 is active.

**Examples:**

```text
min_7d::30      -> hide weekly-thin accounts (below 30% weekly)
min_7d::5       -> hide weekly-exhausted accounts (threshold matches 🟡 boundary)
min_7d::10 min_5h::10 -> combined: both dimensions must meet threshold
```
