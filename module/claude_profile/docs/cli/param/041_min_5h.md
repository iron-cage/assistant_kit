# Parameter :: 41. `min_5h::`

Filters the `.usage` table to hide rows where `5h Left` is below the specified percentage threshold.

- **Type:** `f64`
- **Default:** `0` (no filter)
- **Constraints:** `0` to `100` (inclusive)
- **Commands:** [`.usage`](../command/006_usage.md#command--9-usage)
- **Purpose:** Show only accounts whose 5h Left is at or above the given percentage.
- **Group:** Display Control

**Behavior:** Rows where `5h Left < threshold` are hidden. The threshold is inclusive — a row with `5h Left = threshold` is shown. Rows with `5h Left = —` (no valid quota data, 🔴 accounts) are also hidden when any threshold > 0 is active.

**Examples:**

```text
min_5h::50      -> hide rows where 5h Left < 50%
min_5h::100     -> only show rows with full 5h window
min_5h::15      -> hide hourly-exhausted accounts (threshold matches 🟡 boundary)
```
