# Parameter 064: `id::` — Edge Cases

**Behavioral Divergence Pair:** EC-02 ↔ EC-03 — `id::VALUE` alone activates set mode and writes the preference, exit 0; `id::VALUE reset::1` together is rejected as mutually exclusive, exit 1.

### Test Case Index

| ID | Test | Scenario | Expected | Status |
|----|------|----------|----------|--------|
| EC-01 | `ec1_id_absent_defaults_to_get_mode` | `.model.select` with no `id::` | reads and prints current preference (get mode) | ✅ |
| EC-02 | `ec2_id_present_activates_set_mode` | `id::claude-opus-4-8` | writes `model` to `~/.clr/config.toml`'s user tier; exit 0 | ✅ |
| EC-03 | `ec3_id_and_reset_mutually_exclusive` | `id::claude-opus-4-8 reset::1` | exit 1 — `id:: and reset::1 are mutually exclusive` | ✅ |
| EC-04 | `ec4_id_empty_value_rejected` | `id::` (empty value) | exit 1 — `id:: must be a non-empty model ID` | ✅ |
| EC-05 | `ec5_id_min_boundary_single_char` | `id::x` (shortest non-empty value) | accepted; writes `model = "x"` verbatim; exit 0 | ✅ |
| EC-06 | `ec6_id_max_boundary_long_value` | `id::` with a long realistic model ID string (e.g. `claude-haiku-4-5-20251001`) | accepted; written verbatim, no truncation; exit 0 | ✅ |
| EC-07 | `ec7_id_no_live_validation_against_api` | `id::not-a-real-model-xyz` | accepted at write time — non-empty check only, no live API validation; exit 0 | ✅ |
| EC-08 | `ec8_id_overwrites_existing_pin` | `~/.clr/config.toml` already has `model = "claude-sonnet-5"` → `id::claude-opus-4-8` | overwrites to `"claude-opus-4-8"`; exit 0 | ✅ |
| EC-09 | `ec9_id_preserves_other_prefs_keys` | `~/.clr/config.toml` has unrelated keys alongside `model` | other keys preserved verbatim after write; exit 0 | ✅ |
| EC-10 | `ec10_id_creates_prefs_file_and_dir_if_absent` | `~/.clr/` directory does not exist | `.model.select id::claude-opus-4-8` creates `~/.clr/` and `config.toml`; exit 0 | ✅ |

**Total:** 10 edge case tests

### EC-01: `id::` absent — get mode reads current preference

- **Given:** `~/.clr/config.toml` contains `model = "claude-opus-4-8"`
- **When:** `clp .model.select`
- **Then:** Exits 0. stdout contains `model.select: claude-opus-4-8`. No file is written (read-only get mode).
- **Exit:** 0
- **Source:** [param/064_id.md](../../../../docs/cli/param/064_id.md), [command/007_model.md](../../../../docs/cli/command/007_model.md)

---

### EC-02: `id::VALUE` present — activates set mode, writes preference

- **Given:** `~/.clr/config.toml` absent or does not contain `model`
- **When:** `clp .model.select id::claude-opus-4-8`
- **Then:** Exits 0. `~/.clr/config.toml` contains `model = "claude-opus-4-8"`. stdout contains `model.select: claude-opus-4-8 (pinned)`.
- **Exit:** 0
- **Source:** [param/064_id.md](../../../../docs/cli/param/064_id.md), [command/007_model.md](../../../../docs/cli/command/007_model.md)

---

### EC-03: `id::VALUE reset::1` together — mutually exclusive, rejected

- **Given:** Any `~/.clr/config.toml` state
- **When:** `clp .model.select id::claude-opus-4-8 reset::1`
- **Then:** Exits 1. stderr contains `id:: and reset::1 are mutually exclusive`. `~/.clr/config.toml` is unchanged.
- **Exit:** 1
- **Source:** [param/064_id.md](../../../../docs/cli/param/064_id.md), [param/066_reset.md](../../../../docs/cli/param/066_reset.md)

---

### EC-04: `id::` with empty value rejected

- **Given:** Any `~/.clr/config.toml` state
- **When:** `clp .model.select id::` (empty value after `::`)
- **Then:** Exits 1. stderr contains `id:: must be a non-empty model ID`. `~/.clr/config.toml` is unchanged.
- **Exit:** 1
- **Source:** [param/064_id.md](../../../../docs/cli/param/064_id.md)

---

### EC-05: Minimum boundary — single-character value accepted

- **Given:** `~/.clr/config.toml` absent
- **When:** `clp .model.select id::x`
- **Then:** Exits 0. `~/.clr/config.toml` contains `model = "x"` written verbatim — non-empty is the only validation, no minimum length beyond one character.
- **Exit:** 0
- **Source:** [param/064_id.md](../../../../docs/cli/param/064_id.md)

### EC-06: Maximum boundary — long realistic model ID accepted

- **Given:** `~/.clr/config.toml` absent
- **When:** `clp .model.select id::claude-haiku-4-5-20251001`
- **Then:** Exits 0. `~/.clr/config.toml` contains `model = "claude-haiku-4-5-20251001"` written verbatim, no truncation or reformatting.
- **Exit:** 0
- **Source:** [param/064_id.md](../../../../docs/cli/param/064_id.md)

### EC-07: No live API validation at pin time

- **Given:** `~/.clr/config.toml` absent. No network access assumed.
- **When:** `clp .model.select id::not-a-real-model-xyz`
- **Then:** Exits 0. `~/.clr/config.toml` contains `model = "not-a-real-model-xyz"`. Value is accepted purely on the non-empty check — invalid IDs are only rejected later by the Claude API when `clr` invokes the subprocess, not at pin time.
- **Exit:** 0
- **Source:** [param/064_id.md](../../../../docs/cli/param/064_id.md)

### EC-08: Overwrites an existing pinned value

- **Given:** `~/.clr/config.toml` contains `model = "claude-sonnet-5"`
- **When:** `clp .model.select id::claude-opus-4-8`
- **Then:** Exits 0. `~/.clr/config.toml` contains `model = "claude-opus-4-8"`. Prior value overwritten.
- **Exit:** 0
- **Source:** [param/064_id.md](../../../../docs/cli/param/064_id.md)
