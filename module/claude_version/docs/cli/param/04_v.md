# Parameter :: 4. `v::`

-- **Summary:** Control output detail level.
-- **Type:** `VerbosityLevel`
-- **Default:** 1 (normal)
-- **Commands:** all commands with format:: support
-- **Group:** Output Control

- **Type:** [`VerbosityLevel`](../type/01_verbosity_level.md)
- **Default:** 1 (normal)
- **Validation:** must be 0, 1, or 2; out of range -> exit 1
- **Last-wins:** when repeated, the last value takes effect

```sh
clv.status v::0                       # minimal
clv.status v::2                       # verbose
```
