# CLI Parameter: by

Grouping dimension for aggregate statistics. Controls how events
are bucketed before computing counts, costs, durations, and
success rates.

- **Type:** [`GroupBy`](../type/09_group_by.md)
- **Default:** day
- **Required:** No

```bash
clj .stats by::day                    # Daily breakdown (default)
clj .stats by::model since::30d      # Per-model totals
clj .stats by::command               # Per-command breakdown
clj .stats by::error since::7d       # Error class distribution
clj .stats by::hour since::1d        # Hourly activity today
clj .stats by::creds                  # Per-credential usage
```
