# Test: `ps`

Integration test planning for the `ps` command. See [command/06_ps.md](../../../../docs/cli/command/06_ps.md) for specification.

`ps` lists running Claude Code processes. Tests run in the container where 0 live `claude`
processes are present. The ‚Č•1-session tests spawn a fake `claude` background process using
the `fake_claude_dir` helper and prepend it to PATH.
